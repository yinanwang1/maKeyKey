域名

本来是想申请自己的一个域名，然后再申请一个阿里云的服务器，然后自己搭建博客。多么美好的方案啊。但是到处都是钱，没有钱寸步难行，而且这钱发的有点冤啊。所以就像这有没有方便的。

Git

原来git就提供了这样的方式,用起来，当做是自己的一个简历了。
一个文章在《简书-心情故事》，《简书-技术成长》，《掘金》，《公众号》(码恳垦)再加一个《Git》)吧。

部署

使用hexo来部署，还是很简单。值得一试。
